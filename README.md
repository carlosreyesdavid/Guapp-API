# API GuApp!

## Descripción

API para la gestión de locales

## Datos Técnicos

**Servidor**
- NodeJs

**Base de datos**
- MongoDB

**API URIs**

| URI                      | Método | Descripción                            |
| ------------------------ |:------:| --------------------------------------:|
| /                        | GET    | Aplicación Cliente                     |
| /api/v1                  | GET    | API Root                               |
| /api/v1/locals           | GET    | Listar todos los locales               |
| /api/v1/locals/:id       | GET    | Listar un local por su ID              |
